## Komentorivitoiminnot

Ohjelman saa suoritettua komennolla

```
mvn compile exec:java -Dexec.mainClass=ruokapaivakirja.Main
```

Testit suoritetaan komennolla

```
mvn test
```

Testikattavuusraportti luodaan komennolla

```
mvn jacoco:report
```
