## Komentorivitoiminnot

Jar:in saa luotua komennolla

```
mvn package
```

Jar suoritetaan komennolla

```
java -jar Ruokapaivakirja.jar
```

Testit suoritetaan komennolla

```
mvn test
```

Testikattavuusraportti luodaan komennolla

```
mvn jacoco:report
```

Javadoc luodaan komennolla

```
mvn javadoc:javadoc
```
